# Text Scanner Pro

Инструмент для анализа текста с современным интерфейсом и консольным режимом.

## ✨ Возможности

- 🖥️ **GUI режим**: Современный темный интерфейс на CustomTkinter с поддержкой выбора файлов
- 📟 **Консольный режим**: Легкая версия без графики, работает в любом терминале
- 📊 **Анализ текста**:
  - Количество слов
  - Количество символов
  - Уникальные слова
  - Количество предложений
  - Средняя длина слова
  - Топ-3 самых частых слов
- 📂 **Ввод данных**: Прямой ввод текста или загрузка файла

## 📦 Установка

```bash
pip install customtkinter
```

Python 3.7+, остальные библиотеки входят в стандартную поставку.

## 🚀 Использование

### GUI режим
```bash
python3 main.py
```
Нажмите «📁 Выбрать файл» для загрузки текстового файла или введите текст вручную, затем нажмите «▶ АНАЛИЗИРОВАТЬ».

### Консольный режим
```bash
# Анализ введенного текста
python3 main_console.py

# Анализ файла
python3 main_console.py test_text.txt
```
